# Python招聘数据可视化分析系统 python42

本项目是基于Python技术的招聘数据可视化分析系统，其采用Django框架进行后端开发，使用MySQL作为数据库管理系统，前端则使用了HTML、CSS和JavaScript技术。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：HTML + CSS + JavaScript

## 功能模块

- **用户模块**
  - 登录
  - 注册
- **数据分析模块**
  - 数据统计
  - 各项招聘数据可视化分析
- **个人中心**

## 系统角色

- 普通用户：可以注册、登录，使用数据分析功能。
  
## 运行环境

- Python环境
- Django框架
- MySQL数据库

## 部署步骤

1. 准备Python运行环境。
2. 安装Django框架。
3. 安装MySQL数据库，配置数据库连接。
4. 导入SQL文件，初始化数据库。
5. 运行Django项目。
